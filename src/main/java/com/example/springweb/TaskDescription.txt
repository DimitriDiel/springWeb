Написать микросервис по работе с товаром на складе. (backend)

1. Обеспечить получение и обработку запросов пользователя с клиентской части
   для учета товаров.

2. Обеспечить:
   - хранение данных о товаре (список товаров, категории, их количество и цены)
   - хранение данных о поставщиках
   - приход товара от поставщиков
   - хранение данных о покупателях
   - отгрузка товара покупателям

3. Какие запросы приложение должно уметь обрабатывать:

   Работа с товарами:
    - передать все товары
    - передать все категории
    - передать товары по категории
    - передать товары по категории в ценовом диапазоне
    - передать товары по поставщикам
    - передать товары по категории и поставщику
    - передать данные о количестве конкретного товара
    - показать конкретный товар по id

   Работа с контрагентами:
    - добавление нового поставщика
    - показать список поставщиков
    - добавление нового покупателя
    - показать список покупателей

   Работа с поставками и отгрузками товара:
    - создание прихода
    - показать список приходов
    - создание отгрузки
    - показать список отгрузок
